# ThumbnailChecker

A modern web tool to preview your YouTube thumbnail and title across all major YouTube platforms and devices.

## Features
- Upload a thumbnail and enter a video title
- See how your content appears on:
  - Desktop Home Feed
  - Desktop Search Results
  - Watch Next Sidebar
  - Mobile App (Home & Search)
  - YouTube TV
  - Embedded Player
- Accurate aspect ratios and UI for each platform
- Title length analysis for each platform
- Modern, responsive design

## Usage
1. Open `index.html` in your browser to upload a thumbnail and enter a title.
2. Click the preview button to see your thumbnail on all platforms.
3. Use the back button to return to the upload page.

## Project Structure
- `public/preview.html` — The main preview UI (self-contained)
- `public/styles.css` — (Optional) Additional styles
- `public/app.js` — (Optional) App logic
- `README.md` — This file

## Customization
- Update the HTML/CSS in `preview.html` to match your branding or add more platforms.
- All logic and styles are inline for easy editing.
